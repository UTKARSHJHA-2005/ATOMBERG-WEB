# ATOMBERG FAN WEB
This is the website created for the assignment provided by the Atomberg.<br/>
In this, the user has to input the api and refresh token from its Atomberg Home App.<br/>
<br/>
<strong>Prerequisites</strong><br/>
1. Node.js <br/>
2. Vite <br/>
3. Tailwind CSS <br/>
4. Atomberg Home App <br/>
<br/>
<strong>How to run the app</strong> <br/>
1. Clone the repo<br/>
   ```
   git clone https://github.com/UTKARSHJHA-2005/ATOMBERG-WEB.git<br/>
   ```
   <br/>
2. Install the dependencies<br/>
   ```
   cd ATOMBERG-WEB
   npm install
   ```
3. Run the app (for Vite)<br/>
   ```
   npm run dev
   ```
   Run the app (for node)<br/>
   ```
   node ./src/index.js
   ```
